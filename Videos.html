Diferencia entre software y Hardware



Hadware Software
Definicion Los dispositivos físicos que componen o son accesorios de la computadora Colección de códigos y/o programas que sirven de instrucciones en la computadora.
Interacción Se necesita al menos un software para que el usuario pueda manejar el hardware. Todos los softwares utilizan al menos un componente del hardware.
Función Estructura física que permite la interacción del usuario con el computador. Realizar tareas específicas.
Ejemplos Teclado, pantalla, impresora, ratón, lector de barras. Microsoft Word, Open Office, Photoshop,Linux, MacOS, Windows, Android.
